# 更新说明

## 0.1.0

- 初始化 uni-app + Vue 3.5.x + TypeScript + Vite 6 模板。
- 接入 Wot Design Uni、UnoCSS、Pinia 和 API 自动导入。
- 增加 H5、微信小程序、支付宝小程序、抖音小程序、App、鸿蒙 App 的开发与构建脚本。
- 增加 5 个主页面和自定义底部 Tabbar：首页、我的、组件、示例、其他。
- 增加 Wot Design Uni 组件中心，按官方分类展示并支持点击进入详情示例。
- 增加真实业务示例中心，覆盖基础、初级、中级、高级、复杂五类场景。
- 基础示例新增动态表单验证、输入联动、Toast、Popup、Calendar、DatetimePicker、DropMenu、NoticeBar、Dialog。
- 新增登录页 5 套切换模板：手机号密码、短信验证码、企业租户、管理端、安全快捷登录。
- 新增注册页 5 套切换模板：手机号注册、邀请码注册、企业入驻、创作者开通、App 快速注册。
- 新增插件体系、样式规范、多语言和 HBX 模板说明页面。
- 新增瀑布流示例与内容详情页，支持生活、电商、新闻、朋友圈四种导航形态，电商带商品详情。
- 新增订阅支付模板，支持套餐选择、月/年切换、微信支付、支付宝和银行卡支付方式。
- 新增字母索引联系人、系统消息、评论相关三个业务互动示例。
- 完善评论相关示例，支持一级评论、二级回复、回复二级评论、待审核处理，并修复移动端回复弹层被底部 Tabbar 遮挡的问题。
- 新增可切换动态表单、音乐播放器、歌单列表、文件夹树形结构示例，并完善动态表单的下拉、单选、复杂文本和可增删明细行。
- 增加中文 README、英文 README、License 和 Git 忽略规则。
